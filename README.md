# cli-number-guessing-game-
This is an CLI-based number guessing game.

## Play in terminal

Install dependencies and run the CLI version:

```bash
npm install
node app.js
```

## Web frontend

Open `frontend/index.html` in your browser to play a web version of the same game. The frontend is a static HTML/CSS/JS bundle and does not require a server — just double-click the file or serve the `frontend` folder with any static server.

Example using `http-server` (optional):

```bash
npx http-server frontend -p 5000
# then open http://localhost:5000
```
